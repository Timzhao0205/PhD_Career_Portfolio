# IPRT_02 — Red-team review of ID_02 (recipe-grade tension-profile winding to target Rc)

Reviewed 2026-07-04. All new references fetched this session (3 patents, 3 NPL; fragments/list at bottom).

## 1. Closer prior art the designer missed

- **[NPL-1] Kim, Hahn, Bascuñán, Iwasa et al., IEEE TAS 24(3), DOI 10.1109/TASC.2013.2283855:** NI pancakes wound at 12 N vs 20 N; Rc 783.8 → 160.2 μΩ (~5x time-constant shift); states "**Rc of the NI coil can be predicted from the fixed tension during the winding process**." The tension→Rc relation — the claimed "stored relation" — was published by the canonical NI group 13 years ago. ID_02 §11's rebuttal ("no reference teaches tension as an Rc-setting means") is factually wrong and must be deleted.
- **[NPL-2] Bang, Bradford, Lee, Abraimov (NHMFL/FSU), SUST 37(2) 2024, DOI 10.1088/1361-6668/ad1a46:** coils wound at <1/3/5 kgf; Rc(B) fit functions whose **coefficients are determined by winding force** — a published parametric R̂c(tension) family, from the same institution as [P-017].
- **[NPL-3] Lu, Goddard, Han, Hahn, SUST 2017 (arXiv:1701.00447):** Rc 26-100 μΩ·cm² up to 144 MPa, **decreases with pressure, drifts ~10x under load cycles** — publishes the pressure half of R̂c(tension, pressure) and undermines calibration stability (below).
- **CN102923511B (Shanghai; granted 2015, displays lapsed 2024):** claim 1 recites a 锥度计算模块 (line/curve/**self-learning segmented** taper) + roll-diameter module + fuzzy-PID closed loop — a granted CN claim on programmed diameter-dependent tension trajectories. **CN203071175U** (张力可编程 cell winder, 2013): PLC executes a stored tension-vs-time function against real-time feedback. The ZH 锥度张力 battery-winder shelf ID_02 never searched. **US7997521B1** (2008, displays expired): dancer + load-cell + embedded-controller closed loop, PLC-stored "winding recipes" in spec — E1's retrofit stack, verbatim.

## 2. Obviousness — the existential problem

Claim 1 = [P-009]/[P-010] SC winder + CN102923511B/US8152089B2 programmed-trajectory tension control + [NPL-1]/[NPL-2] tension→Rc relation. Motivation is not hindsight; Kim supplies it in print ("predicted from the fixed tension"). Computing the setpoint by inverting a published curve is automation of a known manual calculation — a classic §103 rationale. What no fetched reference teaches: (i) **spatially graded Rc within one pancake** (search found only non-uniformity *modeling*, no intentional grading); (ii) **co-scheduled normal-force roller keyed to an electrical target**; (iii) defect-map-modulated trajectories. The inventive weight sits in dependents 1-3, not claim 1.

## 3. Enablement / $17.8K realism

The loop closes on **tension**, not Rc; "conforms to the target distribution" rides on feedforward calibration from 24 mini-pancakes at one geometry. Interface pressure in a real coil is set by accumulated hoop-stress history, so R̂c is geometry-dependent — no cross-size validation coil is budgeted. E3's "≥3x inner/outer split by zone-resolved discharge-decay fitting" is an ill-posed inversion from terminal measurements; **weakest BOM assumption: no instrument line for internal voltage taps / zone probes** ($4.5K used nV-meter + source cannot spatially resolve Rc). [NPL-3]'s 10x cycling drift further questions whether an as-wound Rc target persists — 150 m across 2 lots (~4 m/coil, ~25-30 turns) leaves the ±25% scatter gate statistically thin.

## 4. Contamination probe

Four prongs hold: winding machinery is outside the GaN/instrumentation SOW; home-bench + LN2 is credible without lab gear; nV craft is generic EE skill. Condition: the tension→Rc premise must trace to [NPL-1]/[NPL-2], **not** to Hinetics/v0 build experience or unpublished group data — cite them in the notebook. Ironically the same citations that clean prong 3 gut novelty. UIUC agreement check still open (GROUND RULES §2).

## 5. Design-around

Stepwise per-layer setpoints are catchable by drafting ("≥N position-dependent setpoints"). The cheap hole is **externalizing the compiler**: a competitor ships a [P-010]-class winder with CSV schedule import (standard per CN102923511B menus); the fab computes setpoints offline from published curves. The machine never "stores or receives" an Rc target or relation — limitations (a)/(b) vanish. U-012 says buyers treat recipes as *their* moat, so schedule-import is the commercially preferred, non-infringing embodiment.

## 6. Strategic value

Claim 1 as sketched blocks neither TE/Ridgway captive cells nor CN mechanics vendors selling schedule-import winders — near-vanity. Real value: graded-Rc apparatus/article + pressure co-scheduling + per-turn conformance record (W-05 seam), plus defensive filing ahead of the 2027 publication.
