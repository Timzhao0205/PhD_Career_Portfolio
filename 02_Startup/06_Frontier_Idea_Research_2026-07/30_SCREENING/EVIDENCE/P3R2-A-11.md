## 3. Price signals / installed base / bottom-up arithmetic
No public unit price was found for any semiconductor-grade MFC from HORIBA, MKS, Brooks, or Fujikin (all quote-on-request); a generic industrial-distributor price band for non-semiconductor MFCs runs roughly $200-$2,000+ per unit (eBay/distributor listings, **T3/informal, not load-bearing**), which materially understates true semiconductor-grade UHP corrosive-gas MFC pricing (**assumption**: semiconductor-grade MFCs likely run **$1,500-$5,000 per channel**, based on the general step-up in specification/qualification cost between industrial and semiconductor-grade fluid-handling components observed elsewhere in this lane -- **uncertainty: high, no direct quote**).
**$/tool/year throughput-value model (constructed here since none exists in public sources; every step labeled assumption):**
- ALD/ALE tool cycle count per day: **assumption**, 500-2,000 cycles/day for a high-utilization production tool -- **uncertainty: medium**.
- Time saved per cycle if settling drops from an incumbent ~1s to <100ms with zero overshoot (avoiding both slow settling and overshoot-recovery time): **assumption**, 0.3-0.8 seconds saved per gas-switching event -- **uncertainty: high, no direct process data**.
- Switching events per cycle: **assumption**, 2-6 (multiple gas pulses/purges per ALD cycle) -- **uncertainty: medium**.
- Implied time recovered per day: 500-2,000 cycles x 2-6 events x 0.3-0.8s = **300-9,600 seconds/day (5 min-2.7 hours/day)** -- an extremely wide range reflecting compounding uncertainty.
- Tool value per hour of added throughput: **assumption**, $500-$2,000/hour (typical high-end fab-tool amortized-cost-plus-margin range referenced informally in industry discussion, not independently sourced this session) -- **uncertainty: very high, no direct source**.
- **Implied $/tool/year: (5 min to 2.7 hr/day) x 365 x $500-$2,000/hr = roughly $15,000 to $1.4M/tool/year** -- this range is so wide it is not decision-useful without narrowing the two largest uncertainties (actual time saved per event, and actual $/hour tool value), both of which require a real OEM/fab process-engineering data point that was not available this session. **This gap should be treated as still open, exactly as the seed's own FIX language anticipated** ("$/tool/year throughput-value model... to be added in P4") -- the model is now built in skeleton form but not yet evidence-grounded.
- OEM-qualification switching cost: **assumption**, $200k-$1M+ per tool-platform qualification cycle (typical semiconductor-component new-vendor qualification cost order-of-magnitude, not independently sourced) -- likely exceeds the low end of the throughput-value estimate above, which is itself a risk signal for the dual-sourcing thesis.
