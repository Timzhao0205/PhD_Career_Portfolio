# Stage 20 - Three to four full outlines (to subtopic depth)

ultrathink

Read `docs/PROJECT_BRIEF.md` and, if present, `outputs/00_target_journal_brief.md`
and `outputs/10_titles.md`.

Design **3-4 distinct, complete outlines** for the review and write them to
`outputs/20_outlines.md`. Each outline must honor the author's five-part plan
(Introduction; Sensor types + current applications; Future applications;
Potential + standards; Conclusion) with the three middle parts roughly equal, but
the outlines should differ in how they ORGANIZE the middle, so the author can
choose a structure. Suggested distinct strategies (pick 3-4):
- **A - Sensor-family-primary**: Section 2 organized by device family (Hall,
  AMR/GMR/TMR, fluxgate, GMI, SQUID, OPM/SERF, NV-diamond, magnetoelectric,
  MEMS/optical), each with a strengths/weaknesses box and its typical domains;
  applications summarized in a cross-cutting subsection + comparison table.
- **B - Application-primary**: Section 2 organized by domain (energy,
  transportation, industrial, biomed), each pulling in the sensor families that
  serve it; a consolidated device-comparison table up front.
- **C - Capability/《performance-envelope》-primary**: organized by the
  measurement regime (ultra-sensitive pT-scale; precision dc/position; high-
  bandwidth current; imaging/array), mapping families and domains onto each.
- **D - Value-chain / TRL-primary** (business-forward): devices -> systems ->
  deployment -> standards/《market》, threading the four domains through.

For EACH outline provide, to subtopic depth:
- Every section and subsection heading (numbered, MDPI-style), and 1-3 bullets
  under each subsection naming the specific content and the sensor families /
  domains / methods it covers.
- A one-line note of what a figure or table would go there (e.g. "Table 1:
  family x range/resolution/bandwidth/cost/maturity"; "Fig: capability map").
- Rough target length balance (%) across the five parts, keeping parts 2-4 near
  a third each.
- A 2-3 sentence "best when..." note: which author goals/《audience emphasis》this
  structure serves, and its main tradeoff.

End the file with a short **recommendation**: which outline you would pick and
why, and whether any two should be hybridized. Cross-reference title options from
Stage 10 that pair best with each outline. Finish by writing the file.
