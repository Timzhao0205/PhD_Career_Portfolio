raw revenue/cost/profit line items, not copied from any pre-computed ratio page. Two different
pre-computed "financial indicator" pages fetched for this ticker (basic.10jqka.com.cn/002156/finance.html

on the finance.html fetch) are numerically identical to this agent's own independently-computed

the automatic HTML-table-to-text rendering picked the wrong column), not a genuine second net-margin
figure. finance.html is logged as [S435], `"verified":"fetched"` but flagged NOT load-bearing /
superseded; none of its ratio figures are used in the table below. This is why every margin in this
file carries full component arithmetic rather than a single quoted ratio.

Interest-bearing debt = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券(=0):
- FY2023: 385,997.21 + 414,566.87 + 600,258.90 + 0 = 1,400,822.98
- FY2024: 276,596.06 + 449,318.35 + 737,083.76 + 0 = 1,462,998.17
- FY2025: 348,891.41 + 555,257.25 + 972,580.00 + 0 = 1,876,728.66

Net cash = 货币资金 − debt (万元 → CNY million ÷100):
- FY2023: 446,779.36 − 1,400,822.98 = −954,043.62万元 = CNY −9,540.44m
- FY2024: 423,165.01 − 1,462,998.17 = −1,039,833.16万元 = CNY −10,398.33m
- FY2025: 541,008.15 − 1,876,728.66 = −1,335,720.51万元 = CNY −13,357.21m

Narrower alternative (excludes 一年内到期的非流动负债, i.e. only 短期借款+长期借款+应付债券 — the
task's minimum-explicit item list): FY2023 = 446,779.36 − 986,256.11 = −539,476.75万元 = CNY
−5,394.77m; FY2024 = 423,165.01 − 1,013,679.82 = −590,514.81万元 = CNY −5,905.15m; FY2025 =
541,008.15 − 1,321,471.41 = −780,463.26万元 = CNY −7,804.63m. Table above uses the broader
4-component figure because 一年内到期的非流动负债 is itself the current-year-due portion of the same
interest-bearing bank loans and is large (≈40–55% the size of 短期借款 each year); excluding it would
understate TFME's true interest-bearing debt load. 应付债券 = "--" (blank) on the balance-sheet page
in all 3 years, read as "no disclosed bonds-payable balance," not a MISSING data point.
